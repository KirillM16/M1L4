Игра "Покемоны"
Это игра, которая находиться в телеграмм боте.
Имеет интересные функции.
1. Создание покемона(go) (его имя, фото и рост)
2. Получение информации о своих покемонах(info)
3. Атака покемонов других игроков(attack)
4. Кормление своего покемона, позволяющее пополнить количество его жизни(feed)
   
Спасибо за внимание!
